Claude Code, anywhere you have a phone.

Use Claude Code from your phone through the chat apps you already have. The bridge runs locally on your Mac, so nothing goes through a third-party server. Currently Telegram only.

Download for macOS
git clone https://github.com/ykawanabe/pager-for-agents && cd pager-for-agents && ./install.sh

Free & open source · macOS 14+ · Bring your own Claude Code and bun

How it works

Install it on your Mac, pair a Telegram chat, and start messaging Claude. The CLI does the actual work; the menu bar app is just there to make it easier to run.

Per-topic projects

Each Telegram forum topic is pinned to a directory on your Mac, with its own Claude Code session running against that project. Switching topics in Telegram switches projects on the Mac.

Your tools come along

Any MCP server you've added to Claude Code — Notion, browser automation, calendar, mail — is available through the bot too. Same authority you have at the keyboard, just delegating from your phone.

Runs on your Mac

Everything lives on your Mac: the bot token, the allowlist, the paired chats. Claude runs with permissions skipped, so any chat you let through effectively has a shell on your machine. Treat the allowlist accordingly.

Requirements

macOS 14 or newer, Claude Code 2.1.80+, plus bun, a bot token from @BotFather, and your Telegram user ID from @userinfobot.
