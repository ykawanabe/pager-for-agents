Talk to Claude Code from Telegram.

A tiny macOS-native bot that turns Telegram into a real terminal for Claude Code. Per-topic projects, in-chat pairing, MCP outbound — no cloud relay, your Mac stays the trust boundary.

Install on macOS
git clone https://github.com/ykawanabe/claude-telegram-agent && cd claude-telegram-agent && ./install.sh

Free & open source · macOS 14+ · Bring your own Claude Code, bun, tmux

Per-topic project routing

Every Telegram forum topic gets its own Claude Code session pinned to a Mac directory. Switch topics, switch projects — no shared context bleed.

In-chat pairing, host-side trust

Pairing code lives on your Mac; the bot only listens to chats you've claimed. /pair in a new chat (as the paired user) switches in one step — no terminal needed.

MCP-driven outbound

Claude replies via the send_telegram MCP tool, with the chat/thread pinned at spawn time. The bot can't redirect outbound to chats you didn't authorize.

Pager menu bar app

Companion macOS app with status, diagnostics, pairing UI, and per-topic path management. Sleep prevention, live log attach, one-click bot restart.

How it works

Add the bot to any chat → it asks you to send /pair YOUR-CODE → that chat is paired to you. Then send any message: the poller (long-poll getUpdates) dispatches to a per-topic tmux session running claude with an --mcp-config that lets it reply via Telegram. All state lives in ~/.claude-telegram-agent/.

Privacy

No cloud relay. Your bot token, allowlist, mounts, and paired state live in ~/.claude/channels/telegram/ and ~/.claude-telegram-agent/ at mode 600. Telegram talks to your Mac directly. No third-party server in the path.

Security

The bot runs Claude Code with --dangerously-skip-permissions: every tool call is auto-approved. Anyone allowed by your access policy has the same capabilities as Claude on your machine. Use the allowlist; treat the host as trusted; don't share the bot token.

Requirements

macOS 14+, Claude Code v2.1.80+, bun, tmux, a Telegram bot token from @BotFather, your numeric Telegram user ID from @userinfobot.
