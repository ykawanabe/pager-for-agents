Claude Code を、スマホさえあればどこでも。

普段使っているチャットアプリから Claude Code を動かせます。 橋渡しは Mac 上でローカルに動くので、第三者のサーバーは一切経由しません。 現在は Telegram のみ対応。

macOS 版をダウンロード
git clone https://github.com/ykawanabe/pager-for-agents && cd pager-for-agents && ./install.sh

無料・オープンソース · macOS 14+ · Claude Code / bun は各自で用意

仕組み

Mac に入れて、Telegram のチャットをペアリングしたら、あとは Claude にメッセージを送るだけ。 実際の処理は CLI が担い、メニューバーアプリは起動と管理を楽にするためのものです。

トピックごとにプロジェクト

Telegram のフォーラムトピックを Mac 上のディレクトリに紐付け、トピックごとに独立した Claude Code セッションがそのプロジェクトに対して動きます。 Telegram でトピックを切り替えると、Mac 側のプロジェクトも切り替わります。

ツールもそのまま使える

Claude Code に追加した MCP サーバー（Notion、ブラウザ操作、カレンダー、メールなど）は、ボット越しでもそのまま使えます。 キーボードの前にいるときと同じ権限を、スマホから委ねるだけです。

スケジュールで自発的に

チェックリストと時刻を渡せば、Claude が自分で動きます。朝のブリーフィング、監視したいサイト、週次レポート。結果はチャットに届きます。 無人の実行は安全ゲートが読み取りと下書きまでに抑え、リスクのある操作はタップを待ちます。

あなたの Mac で動く

ボットトークン、許可リスト、ペア済みチャット——すべて Mac 上にあります。Claude は権限確認をスキップして動くので、 通したチャットは実質あなたのマシンのシェルを握ることになります。許可リストはそのつもりで設定してください。

必要なもの

macOS 14 以降、Claude Code 2.1.80 以上、加えて bun、 @BotFather のボットトークン、 @userinfobot で取得する Telegram のユーザー ID。
