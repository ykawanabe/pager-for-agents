# agent/MANIFEST.txt — the canonical list of files under agent/ that
# install.sh must ship. Each row:
#
#   <relative-path-under-agent/>	<install-prefix>	<perms>
#
# install-prefix legend:
#   bin    — copied to $BIN_DIR (executables on $PATH)
#   agent  — copied to $INSTALL_DIR/agent/...
#   hooks  — copied to $INSTALL_DIR/hooks/... (read by bot-hooks.json)
#   tpl    — templated by install.sh (sed substitution at install time)
#
# CI test tests/test_install_manifest.sh asserts:
#   1. Every file under agent/ (excluding tests/ and dotfiles) appears here.
#   2. Every entry in this file exists on disk.
#   3. Every entry except `bin` is referenced in install.sh's cp commands.
#
# When you add a new file under agent/ that should ship, add the row HERE
# AND add the cp line in install.sh. The CI test fails fast if either is
# missing.
